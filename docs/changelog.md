# Nhật ký thay đổi (Changelog)

## Phiên bản 1.5 (Tháng 10/2025)
*   **Tính năng đột phá:** **One-prompt App Building** - Cho phép người dùng xây dựng một ứng dụng web full-stack (bao gồm backend, database, frontend) chỉ từ một câu mô tả duy nhất.
*   **Cải tiến:** **Wide Research** - Khả năng chia nhỏ các nhiệm vụ nghiên cứu lớn thành hàng trăm tiểu agent chạy song song để tăng tốc độ và độ bao phủ.
*   **Cải tiến:** **Unlimited Context** - Nâng cấp khả năng xử lý ngữ cảnh dài, cho phép Manus thực hiện các tác vụ phức tạp và dài hơi mà không bị mất thông tin.

## Phiên bản 1.0 (Ra mắt)
*   Giới thiệu các chế độ Chat và Agent.
*   Tích hợp các công cụ cốt lõi (Browser, Code Editor, Terminal).
*   Hỗ trợ kết nối API cơ bản (OpenAI, Google Calendar).
