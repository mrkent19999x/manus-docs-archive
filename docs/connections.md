# Kết nối và Tích hợp

Manus cho phép người dùng kết nối với các dịch vụ bên ngoài thông qua ba phương thức chính:

## 1. Ứng dụng (App Connections)
Đây là các kết nối được tích hợp sẵn và dễ dàng thiết lập thông qua giao diện người dùng.

*   **Gmail/Outlook:** Quản lý email, tóm tắt hộp thư đến, soạn thảo email tự động.
*   **Google Lịch/Outlook Lịch:** Quản lý sự kiện, tìm lịch trống, lên lịch họp.
*   **Notion/ClickUp/Wrike:** Quản lý dự án, tạo nhiệm vụ, cập nhật ghi chú.
*   **GitHub:** Quản lý mã nguồn, tạo kho lưu trữ, theo dõi thay đổi.
*   **Firecrawl:** Thu thập, cào và tìm kiếm dữ liệu web mạnh mẽ.
*   **Zapier/Make/n8n:** Kết nối Manus với hàng ngàn ứng dụng khác để tự động hóa quy trình làm việc.

## 2. API Tùy Chỉnh (Custom API)
Cho phép người dùng kết nối Manus với bất kỳ dịch vụ bên thứ ba nào bằng cách cung cấp Khóa API (API Key) và Base URL.

*   **Mô hình AI:** Tích hợp các mô hình ngôn ngữ lớn như OpenAI (GPT), Google Gemini, Anthropic (Claude) để tăng cường khả năng xử lý.
*   **Dịch vụ Chuyên dụng:** Kết nối với các dịch vụ như Stripe, Cloudflare, Polygon.io, Mailchimp.

## 3. MCP Tùy Chỉnh (Model Context Protocol)
Dành cho các kết nối nâng cao, cho phép Manus tương tác với các máy chủ MCP để truy cập các công cụ chuyên biệt (ví dụ: các công cụ quản lý tài liệu nội bộ).
